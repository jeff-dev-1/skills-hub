# Skills Hub

Public Codex skills that are safe to share outside the internal Milos workspace.

## Skills

- `skills/wechat-editorial-layout`: WeChat public-account editorial layout workflow, including draft validation rules and a lightweight HTML validator.
- `skills/pptx-high-fidelity-reconstruction`: Reconstruct high-fidelity editable PowerPoint decks from screenshots, slide images, PDFs, or visual references using hybrid layering and strict visual QA.
- `skills/icon-finder`: Find, normalize, inspect, and attribute transparent SVG icons for presentations, documents, and visual artifacts.
